hadoopHA

Bash Command Line Utility bundle for Apache Hadoop High Availability (HA) Cluster housekeeping/utility jobs

Utility Details

  • findActiveNameNode.sh :
    Fetch the active NameNode FQDN (fully qualified domain name) in a cluster
    This standalone script will help developers to find the active NN FQDN (not the concise serviceId). It is dynamic so any changes to the cluster configuration are automatically picked up. Takes a maximum of 2 secs. Has configuration options to customize the default port number.
    It is particularly useful to trigger this script before submitting any job that needs the namenode metadata (don't they all do ?) and derive the active NN FQDN upfront to avoid preventable failure/hardcoding.
